<h3 align="center">Qein-X1A-Pro - ESP32-S3 WROOM-1 Development Board</h3>

* Custom ESP32-S3 WROOM-1 based development board with SD Card slot & On-board Programmable RGB LED.

---

## Features

* ESP32-S3-WROOM-1-N16R8 module with Wi-Fi + Bluetooth LE module & AI acceleration.
* WiFi 802.11b, Bluetooth v5.0, 16MB Flash, 384kB ROM, 8MB SRAM.
* SD Card slot for storage expansion.
* Status & debugging LEDs.
* On-board Programmable RGB LED.
* Full GPIO Breakout.
